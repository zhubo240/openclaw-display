春季饮品指南

从早到晚喝对每一杯 · 补水、提神、助眠的全天策略

为什么单独说"喝"？
一个人每天喝下去的液体有 1.5-2 升，但大多数人从不刻意选择喝什么、什么时候喝。 春季湿气重、容易犯困，喝对了能解决一半的春困问题。这份指南把散落在食谱、睡眠、精力管理里的饮品建议集中到一起，给你一份从早到晚的饮品时间表。

一日饮品时间线

6:30 起床
温水一杯 (200-300ml) 水
空腹第一杯水唤醒肠胃。温水（40°C 左右）比冷水温和。可加一片柠檬提神。不要用果汁/牛奶替代。
7:30 早餐
豆浆 / 牛奶 / 低糖酸奶 营养
早餐搭配蛋白质类饮品。自制豆浆优先（春季可加红枣）。避免含糖饮料。
9:00 到岗
第一杯咖啡 咖啡
起床后 90 分钟再喝（→ 咖啡因策略）。 美式/手冲优先。加糖奶 = 额外 100-200 大卡。 日均咖啡因上限 400mg ≈ 2-3 杯。
10:30 上午茶
白水 / 绿茶 水 茶
补水窗口。春季推荐龙井、碧螺春等绿茶（→ 见下方春茶推荐）。或者白开水就好。
13:00 午后
最后一杯咖啡（可选）咖啡
14:00 = 咖啡因截止线。之后的咖啡因会影响当晚睡眠。 如果 13:00 不需要就跳过。
15:00 下午
花茶 / 白茶 / 乌龙（轻发酵）茶
下午犯困但不宜再喝咖啡。茉莉花茶清香提神，咖啡因含量只有咖啡的 1/3-1/4。白茶性质温和。
17:00 下班前
白水 200ml 水
通勤前补水。一天的饮水量到这时应该完成 70-80%。
19:30 晚餐后
花草茶（无咖啡因）花草
菊花、洋甘菊、薄荷、桂花都可以。 不含咖啡因是唯一原则。 → 见下方 6 款花草茶配方。
21:30 收束期
助眠饮品 花草
温牛奶（含色氨酸）或洋甘菊茶。 20:30 后减少饮水量（避免夜间起夜）。 → 与 晚间仪式 收束期配合。

春茶入门

春天是喝新茶的季节。明前茶（清明前采）最鲜嫩。不需要复杂器具，一个玻璃杯就够了。

茶 类型 风味 咖啡因 适合时间 冲泡
龙井 绿茶 豆香、鲜爽 中 上午 80°C · 3g · 150ml
碧螺春 绿茶 花果香、细腻 中 上午 75°C · 3g · 150ml
白牡丹 白茶 清甜、花香 低 全天 90°C · 4g · 150ml
茉莉花茶 花茶 茉莉花香 中低 下午 85°C · 3g · 200ml

不想讲究？最简方案：买一包品质尚可的龙井（100-200 元/50g 足够日常），玻璃杯放 3g 茶叶，80°C 热水冲泡，等 2 分钟。绿茶不要闷盖。

六款花草茶配方

全部无咖啡因，适合下午和晚间。材料在药房或超市都能买到。

春日清肝茶

菊花 5 朵 + 枸杞 10 粒 + 决明子 3g
90°C 热水冲泡 5 分钟。清肝明目，春季养肝首选。

祛湿薏仁水

炒薏仁 15g + 红豆 10g + 陈皮 1 小片
提前泡 2 小时，煮 30 分钟。祛春季湿气，回南天必备。

安神洋甘菊

洋甘菊 2g + 薰衣草 1g + 蜂蜜少许
85°C 冲泡 3 分钟。晚间收束期最佳搭配。

醒脾陈皮茶

陈皮 5g + 山楂 3 片 + 红枣 2 颗
热水焖泡 10 分钟。饭后消食，下午提神。

桂花蜜饮

干桂花 2g + 蜂蜜 1 勺 + 温水 200ml
60°C 温水冲开（水太热破坏蜂蜜）。温胃安神，口感好。

薄荷柠檬水

鲜薄荷 3-5 叶 + 柠檬 2 片 + 冰水/温水
薄荷揉碎释放精油，加柠檬片。下午清爽提神，零咖啡因。

补水计算器

每日饮水量公式
体重(kg) × 30ml = 基础饮水量
60kg → 1800ml / 70kg → 2100ml / 80kg → 2400ml
运动日 +500ml · 干燥天 +300ml · 空调房 +300ml

补水技巧

技巧 说明
小口多次 每次 200ml 左右，一天 8-10 次。不要一次灌 500ml。
桌上放水杯 看见就会喝。透明杯最佳，能看到喝了多少。
关联触发 每次坐下/站起/上厕所 = 喝一口水。不需要记住，行为绑定。
判断脱水 尿液颜色 = 最好的指标。浅黄 = 正常。深黄 = 该喝水了。
晚间减量 20:30 后少喝，避免夜间起夜影响睡眠。

春季饮品避坑

避免 原因 替代
含糖奶茶 高糖 + 反式脂肪。一杯 300-500 大卡，血糖过山车加重困倦 自泡茉莉花茶 + 少许蜂蜜
瓶装果汁 "100% 纯果汁"也是浓缩糖水，缺乏纤维 吃一个完整水果
碳酸饮料 糖 + 磷酸 + 咖啡因。补水效率低 气泡水 + 柠檬片
功能饮料 咖啡因含量不透明，常超标。一瓶可能 = 3 杯咖啡 黑咖啡（咖啡因量可控）
晚间酒精 看似助眠实则破坏深度睡眠。晚间仪式已详述 洋甘菊茶 / 温牛奶
