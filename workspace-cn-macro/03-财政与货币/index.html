地 方 财 政 困 局

收入占 55%，支出占 85%——中间靠转移支付和举债

广义财政四本账（2024年 万亿元） 一般公共预算收入 ~22 税收+非税 政府性基金收入 ~5-6 土地出让金为主（已腰斩） 国有资本经营收入 ~0.7 国企利润上缴 社会保险基金收入 ~11 养老、医疗等 ──────────────────────────────── 合计 ~38-40 万亿 中央 vs 地方的"剪刀差" 收入端：中央 45-47% │ 地方 53-55% 支出端：中央 15% │ 地方 85% 差额：每年约 10万亿 转移支付 + 地方举债 "三保"压力：保基本民生、保工资、保运转 部分县区已出现困难 → 非税收入（罚没等）增长引发争议

政 府 债 务 ：冰 山 结 构

水面以上看似安全，水面以下才是真相

水面以上 — 显性债务（~72万亿 ≈ 54% GDP） ├── 中央国债 ~30 万亿 ├── 地方一般债 ~16 万亿 └── 地方专项债 ~26 万亿 水面以下 — 隐性/准政府债务（~70-85万亿） ├── 城投有息债务 ~40-50 万亿 ← 最大隐患 │ ├─ 城投债（公开市场） ~16-18 万亿 │ └─ 银行贷款+非标 ~25-30 万亿 ├── 政策性银行债 ~25 万亿 └── 政府应付账款等 ~5-10 万亿 ──────────────────────────────────── 总计 ~140-155 万亿（~105-115% GDP） 仅看显性 → "55%，很安全" 加上隐性 → "100%+，全球偏高"
化债 = 借新还旧，缓解但不治本
2023-2024年推出 ~12万亿化债方案：用低利率地方债券置换高利率隐性债务。解决了流动性风险（不会突然爆），但债务总量并未减少。本质是"时间换空间"，最终需要财税改革+经济增长来消化。

5-10%

中小微企业

融资难融资贵的永恒痛点。最需要钱、最难拿到钱。

为什么企业杠杆全球最高？ 1. 融资结构扭曲：间接融资（银行贷款）占80% 每一笔投资都变成债务 → 股权融资（A股）功能弱 2. 国企软预算约束：借钱不怕还不上 政府兜底预期明确 → 银行也愿意贷 3. 投资驱动模式：GDP考核推动持续投资 投资就要借钱 → 杠杆自然上升 国企：借钱容易→利率低→效率差→靠借新还旧 民企：借钱难→利率高→效率高→利润用来还债 → 最需要钱的拿不到，不缺钱的拿太多

货 币 政 策 传 导 失 灵

M2 / GDP = 2.4 倍（全球最高） — 钱印了，但"推不动绳子"

理想的传导链条 央行降息降准 → 银行放贷 → 企业投资 → 就业↑ → 收入↑ → 消费↑ ✓ 实际的传导链条 央行降息降准 → 银行间利率↓ → 但信贷需求不足 ↓ 资金空转：银行买债 → 债券利率↓ → 资金回流 企业套利：借低息贷款 → 买理财产品 居民去杠杆：提前还房贷 ↓ M2↑ 但 信贷脉冲↓ → "宽货币、紧信用" M2增速 vs 名义GDP增速 2010-2015 M2 ~14% 名义GDP ~10% M2/GDP 1.8→2.0x 2016-2019 M2 ~9% 名义GDP ~8% M2/GDP ~2.1x 2020-2024 M2 ~10% 名义GDP ~6% M2/GDP 2.1→2.4x 差距越来越大 → 货币在"空转"而非进入实体经济
"流动性陷阱"特征
降息降准对信贷需求的刺激效果越来越弱。企业不借——看不到投资回报；居民不借——提前还贷。与日本1990年代"推绳子"困境高度相似。货币政策"单打独斗"难以奏效，需要财政政策大力度配合。

居 民 杠 杆 ：分 母 选 错 了

61% 看似安全 → 换成可支配收入一算 → 142%，比美国还高

同一笔债务，不同的分母 口径① 债务/GDP = 82.84/134.9 = 61.4% ← 常被引用 口径② 仅房贷/GDP = 37.68/134.9 = 27.9% ← "看似不高" 口径③ 债务/可支配收入 = 82.84/58.3 = 142% ← 真实偿债压力 为什么差这么多？ GDP 里只有 43% 到了居民手中（美国 76%） 用 GDP 做分母，把政府和企业的收入也算进来了 中美对比 债务/GDP 债务/可支配收入 美国 ~75% ~100% （收入占GDP 76%，差距不大） 中国 ~61% ~142% （收入占GDP 43%，差距巨大） 按实际拿到手的收入计算，中国居民负债率已超过美国

高 储 蓄 + 高 债 务 ：同 一 枚 硬 币

风 险 仪 表 盘

不是急性心梗，而是慢性病——但慢性病也会要命

指标警戒线当前判断
债务增速 vs 名义GDP增速两者接近债务仍快于GDP杠杆仍在上升
利息支出 / GDP<5%~6-8%（含隐性）偿债负担沉重
新增债务中用于还旧债比例<50%部分城投 >80%庞氏特征
企业利润增速 vs 债务增速利润应更快利润↓ 债务↑偿债能力恶化

不会爆的原因

内债为主（95%人民币债务），无外债危机风险；高储蓄率提供融资来源；政府控制金融体系可防挤兑；央行有空间释放流动性；资本管制防止外逃。

慢性病的代价

挤出效应：还旧债没钱投新项目 → 增长放缓。僵尸企业占用资源 → 拖累生产率。财政收入越来越多还债 → 民生被挤压。金融抑制 → 居民存款回报更低 → 消费更弱。最终走向日本化。

核 心 判 断

土地财政已死，新体制未生

土地出让收入三年腰斩，且不可能恢复。消费税下划、房地产税、央地事权改革——每一项都是"牵一发动全身"的改革，但经济下行期推增税改革的政治空间有限。地方财政在"等米下锅"。

债务不是"借太多"，而是"借来的钱没用好"

总量 ~300% 中等偏高但不致命。核心问题是资金配置扭曲：国企拿60%债务贡献40%GDP，民企拿不到钱但效率最高。出路不在"去杠杆"（硬压会引发衰退），而在"调结构"——但这需要国企改革和金融市场化。

货币政策进入"推绳子"阶段

M2/GDP 2.4倍全球最高，但CPI仅0.2%——货币在空转。降息对信贷需求刺激越来越弱，企业不愿投、居民在去杠杆。货币政策已接近极限，必须由财政政策接棒——大规模、直接的需求侧刺激。

接力棒快用完了

过去16年每轮刺激靠换一个部门加杠杆：地方政府→企业→居民→中央政府。三个部门都已杠杆较高。目前只有中央政府资产负债表相对健康——这是最后的子弹，用对了还有机会，用错了可能陷入长期停滞。
