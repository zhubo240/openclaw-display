供需互锁

中国经济最深层的问题：不是"供给不足"也不是"需求不足"，而是供需之间的传导管道多处堵塞。六大结构性错配构成一个互锁的循环系统——单侧刺激治标不治本，需要同时疏通分配、制度和预期三大堵点。

方 法 论 ：为 什 么 单 侧 分 析 失 灵

萨伊定律和凯恩斯都假设"另一侧会自适应"——但中国两边同时卡住了

萨伊定律（供给侧）

"供给创造需求"——造出更好更便宜的东西，消费者自然来买。中国语境："供给侧结构性改革"。

失灵证据：光伏产能1,200GW，全球需求仅500GW。生产出来了，但卖不掉。

凯恩斯（需求侧）

"需求决定供给"——居民有钱就会消费。中国语境："扩大内需"。

失灵证据：居民存款151万亿，但优质养老/医疗/托幼供给严重不足。有钱也花不出去。

"另一侧不适应"的六个证据 ❶ 供给过剩但消费不足 → 萨伊定律失灵 光伏/锂电/电动车产能远超需求 ❷ 有储蓄但无处消费 → 凯恩斯也失灵 151万亿存款 vs 养老/医疗/托幼供给不足 ❸ 技术升级但就业不升级 自动化替代工人→消费下降→产品卖不掉 ❹ 基建投了但回报不来 投资乘数从3x下降至~1.2x，空城空港 ❺ 降息了但没人借 M2增速8% vs CPI 0.2%，流动性陷阱 ❻ 出口强但内需弱 贸易顺差近1万亿$ vs 外需不传导为内需
核心洞察
经济不是两个独立的"侧面"，而是一个循环系统。当前中国的问题是这个循环系统的多个管道同时堵塞了。单纯的供给侧改革（提效率、搞创新）或需求侧刺激（降息、发钱）都只治标。

三 大 结 构 性 堵 点

供给和需求之间的传导管道为什么不通？

供给 ──────→ ✕ ──────→ 需求 │ 结构性堵点 │ ┌─────────────┼──────────────┐ │ │ │ 分配堵点 制度堵点 预期堵点 │ │ │ 钱到不了 想花花不了 不敢花 消费者手里 供给被管制 对未来悲观

分配堵点

劳动报酬占GDP仅52%（发达国家60-65%）。企业留存高分红少，政府收入用于投资而非转移给居民。生产了很多，但劳动者分到的不够买回自己生产的东西。

制度堵点

医疗公立主导民营受限。教育公立不均民营被压。养老市场发育不足。金融投资渠道匮乏。土地住房供给端管制。有需求的领域，供给被行政管制卡死。

预期堵点

居民预期收入下降→不消费。企业预期需求下降→不投资。银行预期不良上升→不放贷。地方预期土地卖不掉→不开工。每方的"理性"行为合成了集体的"非理性"结果。

互 锁 的 循 环

六大错配不是独立的，它们构成一个自我强化的恶性循环

六大错配的互锁循环 制造业过剩 ←──┐ │ │ ↓ │ 价格下跌 投资错位 （通缩） （钱去了不该去的地方） │ │ ↓ │ 企业利润↓ │ │ │ ↓ │ 劳动力技能错配 → 就业困难 → 收入不足 │ ↓ 储蓄困在银行 （投资渠道差） │ ↓ 消费不足 + 服务业短缺 （想花的地方供给不够） │ ↓ 需求不足 → 制造业更过剩 ↑ 回到起点 ──→ 循环加速
这不是哪一侧的问题
这是整个循环系统的管道堵塞。每一处堵点都在加重其他堵点。供给过剩压低价格→企业降薪裁员→消费更弱→需求更不足→供给更过剩。打破循环需要同时在多个堵点上动手术，而每一项都涉及深层利益重新分配。

核 心 判 断

判断 01
中国经济的核心问题是"供需传导管道的多处堵塞"而非简单的供给或需求不足
单看任何一侧都会误诊：供给侧看到产能过剩以为要去产能，需求侧看到消费不足以为要发钱。但问题的根源是两侧之间的传导机制——分配、制度、预期——同时出了问题。生产者造的东西消费者买不起或不需要，消费者想要的东西没人造或被管制挡住了。
判断 02
六大错配互相强化，构成自锁循环——没有单一突破口
制造业过剩→价格战→企业降薪→消费不足→需求更弱→制造业更过剩。劳动力技能错配→就业困难→收入下降→储蓄困在银行→资本市场弱→创新融资难→产业升级卡壳。每处堵点都在加重其他堵点，没有哪一处是独立的。
判断 03
真正需要的是"疏通传导机制"的结构性改革，而非刺激政策
分配改革（提高劳动报酬占比）、制度改革（放开服务业管制、户籍改革）、市场化改革（让价格信号发挥作用）——这三类改革才是治本之策。但每一项都涉及深层利益的重新分配：劳动报酬上升意味着企业利润和政府税收下降；放开服务业管制意味着国有机构让出空间；户籍改革意味着城市居民分享公共资源。
判断 04
改革之所以难推，恰恰因为它需要同时在多个利益集团的奶酪上动刀
这是一个政治经济学问题而非纯经济学问题。技术上知道该怎么做（提高劳动报酬、放开管制、改革户籍、完善社保），但每一步都面临强大的既得利益阻力。渐进式改革的时间窗口正在收窄——人口老龄化、外部环境恶化、债务压力上升都在要求更快的行动。
