Q12: 如何从模型预测到实际持仓？

模型输出的是每只股票的预测得分（alpha signal），但从得分到真正的持仓还需要经过选股、定权、约束检查和交易指令生成等多个环节。

1. TopkDropout（Qlib 默认策略）

每期持有预测得分最高的 topk 只股票，通过 n_drop 参数限制每期的最大换手数量。

参数含义推荐值（个人）说明
topk持仓股票数10 ~ 15百万级资金建议 10-15 只
n_drop每期最多换出数2 ~ 3控制换手率，避免每期大幅调仓
Step 1
预测得分排序
Step 2
标记现有持仓
Step 3
保留 top(k+n_drop)
Step 4
替换末尾持仓
Step 5
输出新持仓

TopkDropout 的优缺点 优点：实现简单，换手率可控，不需要额外优化器。缺点：完全忽略行业/风格暴露，等权分配不考虑信号强度差异。适用：个人量化入门首选。

2. 等权 / 市值加权 / IC 加权

加权方式原理优点缺点
等权每只股票分配相同资金最简单，无需额外数据受小盘股波动影响大
市值加权按流通市值占比分配接近指数特征，波动低因子暴露被大盘股稀释
IC 加权按模型预测得分分配集中资金于高信号股风险集中，波动大
 
import numpy as np
def calc_weights(scores, method="equal", market_caps=None):
    if method == "equal":
        return np.ones(len(scores)) / len(scores)
    elif method == "market_cap":
        return market_caps / market_caps.sum()
    elif method == "ic_weighted":
        pos = scores - scores.min() + 1e-6
        return pos / pos.sum()

3. 优化器方法（Mean-Variance / Risk Parity）

max( alpha'w - lambda * w'Sigma*w )  s.t.  sum(w) = 1, w >= 0

均值-方差优化需要协方差矩阵估计（历史60~120日样本协方差或 Ledoit-Wolf 收缩）。风险平价让每个持仓贡献等量风险，不需要 alpha 估计。

import cvxpy as cp
def mean_variance_optimize(alpha, cov_matrix, risk_aversion=1.0):
    n = len(alpha)
    w = cp.Variable(n)
    ret = alpha @ w
    risk = cp.quad_form(w, cov_matrix)
    objective = cp.Maximize(ret - risk_aversion * risk)
    constraints = [
        cp.sum(w) == 1,   # 满仓
        w >= 0,            # 不做空（A股限制）
        w <= 0.10,         # 单股上限 10%
    ]
    prob = cp.Problem(objective, constraints)
    prob.solve(solver=cp.ECOS)
    return w.value
A 股特殊约束 整手交易（100股倍数）、涨跌停不可交易、T+1 制度（当天买入次日才能卖出）。

4. 实际决策流程

模型预测得分 → 排序选股 → 确定权重 → 检查约束 → 目标持仓 → 对比现有 → 交易指令 → 成本评估 → 过滤小额 → 最终执行
务实建议 初期用 TopkDropout + 等权即可。优化器方法等模型信号稳定后再考虑引入。

Q13: 行业暴露和风格暴露怎么控制？

未控制的因子暴露等于不可控的风险。组合不知不觉集中在某个行业或风格上，市场风格切换时就可能遭受巨大回撤。

1. 什么是因子暴露

因子暴露是组合相对于基准在各风格因子上的偏移量。例如组合平均 PE=15 而基准 PE=20，则有正向 value 暴露。

2. 行业暴露控制

方法原理难度效果
硬约束法每个行业权重偏离基准不超 ±5%中严格中性
行业内选股在每个行业内分别排序选 Top N低天然中性

4. 暴露计算与约束代码

import numpy as np, pandas as pd

def calc_exposure(portfolio_df, benchmark_df, factor_col):
    """计算组合相对基准的因子暴露"""
    port_mean = portfolio_df[factor_col].mean()
    bench_mean = benchmark_df[factor_col].mean()
    return (port_mean - bench_mean) / benchmark_df[factor_col].std()

def apply_constraints(weights, industry_labels, max_industry=0.20, max_stock=0.10):
    """应用仓位约束"""
    weights = np.minimum(weights, max_stock)  # 个股上限
    for ind in set(industry_labels):
        mask = np.array(industry_labels) == ind
        ind_w = weights[mask].sum()
        if ind_w > max_industry:
            weights[mask] *= max_industry / ind_w
    return weights / weights.sum()  # 归一化

2. 冲击成本经验模型

impact = k × sqrt( V / ADV )

V=交易量，ADV=日均成交量，k=经验系数（A股约0.1~0.3）。小资金（<500万）冲击成本基本可忽略。

import numpy as np
def estimate_impact(trade_value, adv, k=0.2):
    """估算冲击成本（平方根模型）"""
    return k * np.sqrt(trade_value / adv)

def total_cost(trade_value, direction, adv, commission=0.00025, stamp=0.0005):
    """计算总交易成本"""
    cost = commission  # 佣金（单边）
    if direction == "sell": cost += stamp  # 印花税仅卖方
    cost += estimate_impact(trade_value, adv)
    return cost

# 示例：100万买入日均成交5亿的股票
print(f"冲击: {estimate_impact(1e6, 5e8, 0.15):.4%}")  # ~0.67bp，可忽略

4. 降低换手率的实用技巧

技巧原理效果
n_drop参数TopkDropout限制每期最大换手数直接控制，效果显著
买卖阈值分离进入topk才买，跌出top(k+buffer)才卖减少边界股反复进出
交易成本惩罚选股得分中扣除预期交易成本自动倾向保留现有持仓
最小交易额过滤忽略调仓金额<5000元的交易避免小额无效调仓
 
def adjust_score_with_cost(scores, holdings, cost=0.006):
    """交易成本惩罚：不在持仓中的股票扣减预期成本"""
    adjusted = scores.copy()
    for s in adjusted.index:
        if s not in holdings: adjusted[s] -= cost
    return adjusted

def hysteresis_filter(scores, holdings, topk=15, buffer=5):
    """买卖阈值分离：买入要进top k，卖出要跌出top(k+buffer)"""
    ranked = scores.sort_values(ascending=False)
    keep = [s for s in holdings if s in ranked.index[:topk+buffer]]
    for s in ranked.index:
        if len(keep) >= topk: break
        if s not in keep: keep.append(s)
    return keep[:topk]

Q15: 仓位管理和止损规则怎么设？

仓位管理不会帮你找到好股票，但能在犯错时保护你不至于血本无归。

1. 个股仓位上限

持仓数（topk）等权权重推荐上限绝对上限
1010.0%20%15%
156.7%13%15%
205.0%10%15%
303.3%7%10%
推荐个股上限 = min( 1/topk × 2, 15% )
A 股集中持仓的额外风险 跌停板 -10%（创业板/科创板 -20%）且无法卖出。单股仓位 15% 连续跌停 3 天，组合直接回撤 4.5%+。

2. 总仓位控制

策略仓位优点缺点
满仓100%最大化因子暴露承担全部系统性风险
固定留现金70-80%保留流动性缓冲牛市中拖累收益
均线择时动态回避大级别熊市震荡市反复止损
 
def market_timing_position(index_close, ma_period=200):
    """200日均线上方满仓，下方降到60%"""
    ma = index_close.rolling(ma_period).mean()
    return 1.0 if index_close.iloc[-1] > ma.iloc[-1] else 0.6

def smooth_position(index_close, ma_period=200):
    """渐变仓位：偏离均线幅度线性调整"""
    ma = index_close.rolling(ma_period).mean()
    dev = (index_close.iloc[-1] - ma.iloc[-1]) / ma.iloc[-1]
    return max(0.5, min(1.0, 0.75 + dev * 2.5))
初期建议 建议满仓运行：(1) 减少决策维度；(2) 择时本身是另一个难题；(3) 长期市场向上，满仓机会成本更低。

3. 止损规则

止损类型条件动作推荐
个股止损跌超买入价 15-20%强制卖出不推荐
组合止损回撤超 10-15%减仓 50%推荐
时间止损持仓超 N 期仍亏强制卖出不适用

因子策略通常不设个股止损：模型每期重新评估，差的股票会自动被调出；止损常被"洗出"好股票；频繁止损增加交易成本。推荐组合级止损保护总资产。

class PortfolioStopLoss:
    def __init__(self, max_dd=0.12, reduce=0.5, recover=0.05):
        self.max_dd = max_dd        # 止损回撤阈值
        self.reduce = reduce        # 减仓比例
        self.recover = recover      # 恢复阈值
        self.peak = 0; self.stopped = False

    def update(self, nav):
        self.peak = max(self.peak, nav)
        dd = (self.peak - nav) / self.peak
        if dd >= self.max_dd and not self.stopped:
            self.stopped = True
            return self.reduce
        if self.stopped and dd < self.recover:
            self.stopped = False
            return 1.0
        return self.reduce if self.stopped else 1.0

4. 资金管理的 Kelly 准则

f* = expected_return / variance = mu / sigma^2  （实践中用半 Kelly: f*/2）
def kelly_fraction(expected_return, volatility, half=True):
    f = expected_return / (volatility ** 2)
    if half: f *= 0.5
    return max(0, min(1.0, f))

class RiskMonitor:
    def __init__(self, nav): self.nav = nav
    def max_drawdown(self):
        peak = self.nav.cummax()
        return ((self.nav - peak) / peak).min()
    def daily_report(self, holdings):
        r = {"nav": self.nav.iloc[-1], "ret": self.nav.pct_change().iloc[-1],
             "mdd": self.max_drawdown(), "max_w": holdings["weight"].max(),
             "top_ind": holdings.groupby("industry")["weight"].sum().max()}
        alerts = []
        if r["ret"] < -0.03: alerts.append("日跌幅超3%!")
        if r["mdd"] < -0.10: alerts.append("回撤超10%! 考虑止损")
        if r["max_w"] > 0.15: alerts.append("单股仓位超15%!")
        if r["top_ind"] > 0.25: alerts.append("单行业超25%!")
        r["alerts"] = alerts; return r
组合构建与风控总结 (1) 从 TopkDropout + 等权开始，最简单可靠的起步方案。
(2) 行业内选股天然实现行业分散，无需复杂优化器。
(3) 准确建模交易成本（佣金+印花税+冲击），月换手率控制在 20-40%。
(4) 组合级止损保护总资产，不设个股止损，让模型自动管理进出。
(5) 建立每日/周/月三级监控体系，数据驱动风控决策。
