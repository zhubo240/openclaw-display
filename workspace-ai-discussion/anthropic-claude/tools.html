Claude Code
Claude Code
2025-05 发布 · 终端原生 AI 编程助手
Anthropic 官方 CLI 编程工具，直接在终端中运行。能读写文件、执行命令、搜索代码、管理 Git，支持多轮对齐式任务规划。随 Claude 4 同步发布，成为 Anthropic 收入增长的核心驱动。
核心产品CLI
Claude Code 关键特性
持续迭代中
权限系统：分级权限控制（允许/拒绝/始终允许），沙盒化 Bash 执行。 Hooks：可配置的事件钩子，在工具调用前后执行自定义命令。 Agent SDK：基于 Claude Code 构建自定义 AI 代理的 SDK。 IDE 集成：VS Code / JetBrains 扩展，内嵌 Claude Code 面板。 Git 集成：自动 diff 分析、commit message 生成、PR 创建。
PermissionsHooksAgent SDK
CLAUDE.md 上下文系统
项目级 AI 记忆
通过项目根目录的 CLAUDE.md 文件为 Claude Code 提供持久化上下文（编码规范、架构说明、常用命令）。支持层级覆盖：全局 ~/.claude/CLAUDE.md → 项目 → 子目录。

MCP 协议
Model Context Protocol (MCP)
2024-11 发布 · 开放标准
AI 模型与外部工具/数据源的标准化连接协议。类似 AI 的「USB-C」— 一个协议连接所有工具。采用 JSON-RPC 2.0，支持 stdio 和 SSE 两种传输方式。
开放标准JSON-RPC
MCP 生态系统
快速增长中
官方 Server：GitHub、Google Drive、Slack、Postgres、Puppeteer 等。 社区 Server：数百个第三方实现覆盖 Notion、Jira、AWS、Kubernetes 等。 SDK：TypeScript、Python、Java、Go 官方 SDK。 2025-11：发布 MCP 1.0 正式版，新增 OAuth 2.1 认证和可发现性协议。

API 与平台
Messages API
核心 API
统一的消息格式 API，支持多轮对话、系统提示词、流式输出。支持工具调用（tool use）、视觉输入、PDF 解析。通过 Anthropic 直接 API、AWS Bedrock、Google Vertex AI 三个渠道提供。

Extended Thinking API
2025-02 起
允许模型在回复前进行深度推理的 API 参数。开发者可设置 thinking budget 控制推理深度。思考过程默认加密返回（支持缓存），可选明文显示。

Claude Agent SDK
