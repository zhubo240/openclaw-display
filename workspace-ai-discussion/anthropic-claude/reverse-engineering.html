上下文管理与压缩机制
2025 · 社区分析
发现 Claude Code 在接近上下文限制时自动压缩早期对话（auto-compaction）。压缩通过生成对话摘要替换原始消息实现。CLAUDE.md 文件内容始终保留在上下文中不被压缩。

MCP Server 安全审计
2025 · 安全社区
社区发现多个第三方 MCP Server 存在安全隐患：未验证的工具调用、SQL 注入风险、过宽的文件系统访问权限。推动了 MCP 1.0 中 OAuth 2.1 认证和 scope 限制的引入。
