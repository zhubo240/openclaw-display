## 核心议题

### 大宗商品超级周期理论
- 什么是大宗商品超级周期？历史上有几轮？每轮持续多久？
- 1850年至今的大宗商品真实价格走势（扣除通胀后）

- 高盛/世界银行对超级周期的学术定义和识别标准

### 历史上的超级周期
1. **1900-1920s**：第一次工业化高峰 + 一战
2. **1940-1950s**：二战 + 战后重建
3. **1970s**：石油危机 + 布雷顿森林体系崩溃 + 滞胀
4. **2000-2011**：中国工业化 + 城镇化 + 基建狂潮
5. **2020s-?**：当前是否正在进入第五轮超级周期？

### 当前大宗商品形势
- 金银暴涨：黄金突破3000美元，白银跟涨
- 铜价创新高：AI数据中心 + 电动车 + 电网升级的需求叠加
- 原油：地缘冲突（中东、俄乌）vs 新能源替代的拉锯
- 农产品：极端天气 + 地缘风险
- 铝、锂、钴、稀土：新能源转型的关键矿产

### 本轮涨价的驱动力
1. **供给侧**：过去10年资本开支不足（ESG约束、矿业投资低迷）
2. **需求侧**：AI算力基建（电力+铜）、绿色转型（锂+铜+银）、印度工业化
3. **货币因素**：全球央行印钞、美元信用松动、去美元化→黄金
4. **地缘政治**：资源民族主义（印尼镍出口禁令）、供应链武器化
5. **通胀预期**：大宗商品作为通胀对冲资产的重新定价

### 关键争论
1. 我们真的进入了第五轮超级周期，还是只是周期性反弹？
2. 新能源转型是推高大宗（铜锂银需求）还是压低大宗（减少石油需求）？
3. AI对大宗商品的净影响：增加电力/算力需求 vs 提高资源利用效率
4. 中国需求减速（房地产下行）能否被印度/东南亚接棒？
